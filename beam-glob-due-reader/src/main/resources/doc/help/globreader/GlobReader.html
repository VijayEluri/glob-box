Glob DUE Reader

Supported Products

The Glob DUE Reader module provides the capability of reading data products provided by multiple Glob DUE projects. This module is the inheritor of the former GlobToolbox. Since most functionality of the Toolbox has been moved into BEAM itself this set of reader has left over. The following table comprises the supported products:

Project Description
GlobCover The Bimonthly/Annual MERIS FR mosaics and the Global/Regional Land Cover are supported. For more information regarding the GlobCover data please visit GlobCover web site (http://due.esrin.esa.int/globcover/).
GlobColour GlobColour Level-3 data products are provided in NetCDF-3 file format. They can be obtained in two variations. There are the binned (ISIN grid) and mapped (Plate Caree) data products. For additional information on GlobColour data products please visit the GlobColour web site (www.globcolour.info).
ATSR World Fire The ATSR World Fire Atlas provides global fire maps from june 1995 to now. Both sensors (ATSR2 and AATSR) are supported. For additional information about the project please visit the ATSR World Fire Atlas web site (http://due.esrin.esa.int/wfa).
Medspiration The Medspiration project combines sea surface temperature (SST) data measured independently by several different sensors. The resutling L2 Pre-Processed (L2P) and the Analysed SST (L4) HR-DDS data products are supported. For additional information about the project please visit the Medspiration web site (www.medspiration.org).
GLC 2000 The Global Land Cover 2000 project provides a harmonized land cover database over the whole globe. The output of this project is supported is supported in the GeoTIFF format. For additional information about the project please visit the GLC 2000 web site (http://bioval.jrc.ec.europa.eu/products/glc2000/glc2000.php).
GlobAerosol The project provides a global aerosol data set from merging ATSR-2, AATSR, MERIS and SEVIRI satellite data. The data set spans a period from 1995 to 2007. The following product types are supported:
o Individual-Sensor Orbit Aerosol - All sensors, except AATSR
o Individual Global Aerosol
o Merged Global Aerosol
o Statistical Products
For additional information about the project please visit the GlobAerosol web site (www.globaerosol.info/).
IGBP Global LCC The Global Land Cover Characteristics provides a classification of the land. For more information about the project please visit the GLCC web site (http://edc2.usgs.gov/glcc/glcc.php) or the IGBP web site (www.igbp.net).

 

The products can simply be opened by using File -> Open Product.... The correct product type will be detected automatically. The products can also be opened by selecting the product type explicitly within File -> Import Raster Data -> Glob DUE Data.
