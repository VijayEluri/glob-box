About the BEAM GlobToolbox Reader Pack, Version 1.0

Introduction

The BEAM GlobToolbox project has been brought into life in order to support users of data acquired by ESA's Glob* projects.

The GlobToolbox is developed by Brockmann Consult under contract to ESA.

Capabilities

Supported Products in GlobToolbox Reader Pack 1.0

Version 1.0 of the BEAM GlobToolbox is capable of reading data products provided by multiple Glob* projects. The following table comprises the supported products and also the products which are not or not fully supported:

Project Product Type Support
GlobCover Bimonthly/Annual MERIS FR mosaics Supported
Global/Regional Land Cover Supported but additional files like "Globcover_Legend.xls", "Globcover_Preview.gif" and color map file in ArcInfo, ArcView, and Envi format are not supported
GlobColour Global/DDS binned product Supported
Global/DDS mapped product Supported
MERIS L3 product Supported already in BEAM
GlobCarbon All types Not yet supported
ATSR World Fire ATSR2 and AATSR Supported
Medspiration L2 Pre-Processed (L2P), Analysed SST (L4) HR-DDS Data Supported
Global Land Cover 2000 GLC2000 Supported as GeoTIFF, but not data in ESRI format
GlobAerosol Individual-Sensor Orbit Aerosol All sensors supported except AATSR
Individual Global Aerosol Supported
Merged Global Aerosol Supported
Statistical Products Supported
IGBP Global LLC Not yet Supported
